# Player Meta Template

## Purpose
Current meta analysis and tier lists.

## Variables
- `{{game}}` — Game name
- `{{patch}}` — Patch version
- `{{format}}` — Ranked/casual/tournament

## Response Structure

### 1. Meta Snapshot
```
📊 Meta Snapshot — Patch [Version]
Overall State: [Stable/Shifting/Volatile]
Last Major Change: [What caused shift]
```

### 2. Tier List
```
**S-Tier (Must Pick/Ban):**
- [Character 1]: [Why dominant]
- [Character 2]: [Why dominant]

**A-Tier (Strong Picks):**
- [Character 3]: [Strengths]
- [Character 4]: [Strengths]

**B-Tier (Viable):**
- [List]

**C-Tier (Situational):**
- [List]

**D-Tier (Avoid):**
- [List]
```

### 3. Key Trends
What's rising/falling in popularity.

### 4. Patch Impact
How recent changes affected the meta.

### Example: Teamfight Tactics

```
📊 TFT Set 10 Meta — Patch 14.6
Overall State: Shifting | Last Major Change: Heartsteel nerfs

**S-Tier Comps:**
1. **Pentakill Vertical** — Still broken after minor nerfs
   - Core: Kayle 3*, Mordekaiser, Viego
   - Why: Frontline unkillable, backline melts everything

2. **Country Samira** — Rising after buffs
   - Core: Samira 3*, Urgot, Tahm Kench
   - Why: Samira executes reset chain

**A-Tier Comps:**
- Disco TF: Consistent top 4
- Jazz MF: High roll potential
- K/DA Ahri: Needs perfect items

**Key Trends:**
⬆️ Rising: Country (Samira buff), Jazz (flexible)
⬇️ Falling: Heartsteel (nerfed), True Damage (countered)

**Patch Impact:**
- Heartsteel cashout nerfed from 8% → 6% → less econ
- Samira AD ratio buffed → now viable carry
- Disco ball HP reduced → slightly weaker frontline
```
