# Player Build Template

## Purpose
Character/agent build optimization and gear selection.

## Variables
- `{{game}}` — Game name
- `{{character}}` — Character/agent name
- `{{role}}` — Role/playstyle
- `{{patch}}` — Patch version

## Response Structure

### 1. Build Header
```
🎮 [Character] — [Role] Build
📊 Patch: [Version] | Win Rate: [%] | Pick Rate: [%]
```

### 2. Core Items/Loadout
List in priority order with explanations:
```
**1. [Item Name]** — [Why it's core]
   - Stats: [Key stats provided]
   - Synergy: [How it works with character kit]
   - Timing: [When to buy/build it]
```

### 3. Skill Order
```
**Max Order:** Q → E → W → Q → Q → R
**Why:** [Explanation of priority]
```

### 4. Runes/Talents/Perks
```
**Primary Tree:** [Tree Name]
- [Keystone]: [Why]
- [Major 1]: [Why]
- [Major 2]: [Why]
- [Minor 1]: [Why]
```

### 5. Matchup Tips
```
**Strong Against:** [List]
- Why: [Explanation]
- Playstyle: [How to exploit]

**Weak Against:** [List]  
- Why: [Explanation]
- Counterplay: [How to survive]
```

### 6. Power Spikes
```
**Early Game (Levels 1-5):** [Strength]
**Mid Game (Levels 6-13):** [Strength]
**Late Game (Levels 14+):** [Strength]
```

### Example: League of Legends

```
🎮 Camille — Top Lane Split Push
📊 Patch: 14.6 | Win Rate: 51.2% | Pick Rate: 8.7%

**Core Items:**
1. **Divine Sunderer** — Sheen proc + %HP damage synergizes with Q2 true damage
2. **Ravenous Hydra** — Wave clear for split pushing, omnivamp for sustain
3. **Death's Dance** — Survive burst in teamfights when you need to join

**Skill Order:** Q → E → W → Q → Q → R
**Why:** Max Q for trading power, E second for mobility/gank setup

**Runes:**
**Primary (Precision):**
- Conqueror: Stack AD for Q2 nuke
- Triumph: Survive tower dives
- Legend: Alacrity: Faster Q animations
- Last Stand: Low HP = more damage

**Matchup Tips:**
**Strong vs Jax:** Your E outranges his counterstrike
**Weak vs Malphite:** Rush Mercury's Treads, play for teamfights

**Power Spikes:**
- Level 1: Strong with Q trades
- Level 2: E engage potential
- Sheen: First major spike
- Divine Sunderer: Can 1v1 most tops
```
