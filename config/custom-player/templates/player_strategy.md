# Player Strategy Template

## Purpose
Game strategy, tactics, and improvement guides.

## Variables
- `{{game}}` — Game name
- `{{situation}}` — Situation (early game, teamfights, etc.)
- `{{skill_level}}` — Player skill level

## Response Structure

### 1. Situation Analysis
What makes this situation challenging/important.

### 2. Core Principles
2-3 fundamental concepts that apply.

### 3. Step-by-Step Guide
Concrete actions to take.

### 4. Common Mistakes
What most players get wrong.

### 5. Pro Tips
Advanced techniques from high-level play.

### Example: Valorant Attack Side

```
**Situation:** Attacking Ascent A Site
**Difficulty:** Medium | **Key Concepts:** Map control, utility usage, timing

**Core Principles:**
1. Control mid before committing to site
2. Use utility to clear common angles
3. Trade kills — don't peek alone

**Step-by-Step:**
1. **Prep Phase:** Buy rifles + smokes + flashes
2. **Early Round:** 
   - 2 players push mid (controller + duelist)
   - 1 holds catwalk
   - 2 push A main slowly
3. **Execute:**
   - Smoke heaven + tree
   - Flash into site
   - Clear gen, then default, then site
4. **Post-plant:**
   - 1 heaven, 1 gen, 1 tree, 2 fall back

**Common Mistakes:**
- Rushing A main without mid control (get flanked)
- Not smoking heaven (die to OP)
- All stacking site (lose map control)

**Pro Tips:**
- Jett can dash onto generator for unexpected angle
- Sova recon dart can clear heaven from A main
- Save one flash for retake attempt
```
